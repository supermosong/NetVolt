What is a Computer?

A computer is a machine that takes in information, processes it using a set of instructions, and produces a result — all in a split second.

Think of it like a very fast chef. You give the chef ingredients (your input), the chef follows a recipe (the program), and serves you a meal (the output). Every computer — from a giant server to your smartphone — works on exactly the same idea.

Key idea

Every computer — no matter its size — does three things: Input → Process → Output. Keep this in mind as you learn about the parts below.

Inside a Computer

The diagram below shows the main parts found inside a desktop computer. Hover over or tap any part to learn more.

CPU

The Central Processing Unit is the brain. It carries out every instruction a program gives — billions of times per second.

RAM

Random Access Memory is your computer's short-term memory. It holds programs you have open right now — and clears when you shut down.

Storage

SSD or HDD — this is where your files, photos, and programs live permanently, even when the power is off.

GPU

The Graphics Processing Unit handles everything you see on screen — from basic text to complex 3D game graphics.

Input & Output Devices

Devices that send information into the computer are called input devices. Devices that send information out to you are called output devices.

A keyboard lets you type words — that's input. The monitor shows those words — that's output. Some devices do both (a touchscreen, for example, receives your taps and displays images).

Storage — RAM, SSD, and HDD

Computers use three very different types of storage, each with a different job, speed, and lifespan.

Imagine your computer is a chef's kitchen. RAM is the countertop — it's small but you can grab things instantly. The SSD is the fridge — faster to reach and holds more. The HDD is the pantry — slower but holds huge amounts of food cheaply.

Fastest Temporary

RAM

Random Access Memory. Holds programs and data currently in use. Blazing fast but clears completely when you turn the computer off.

  • Typical size: 8 GB – 64 GB
  • Speed: Extremely fast
  • Keeps data when off: No
Fast Permanent

SSD

Solid State Drive. Uses flash chips (like a giant USB stick) to store your files permanently. Much faster than an HDD and has no moving parts.

  • Typical size: 256 GB – 4 TB
  • Speed: Very fast
  • Keeps data when off: Yes
Slower Permanent

HDD

Hard Disk Drive. Uses spinning magnetic disks to store data. Slower than an SSD but much cheaper per gigabyte — great for bulk storage.

  • Typical size: 1 TB – 20 TB
  • Speed: Moderate
  • Keeps data when off: Yes

Common mix-up

People often confuse RAM with storage. Remember: RAM is temporary (your desk) — storage is permanent (your filing cabinet). More RAM means more programs open at once; more storage means more files saved.

Software vs Hardware

Hardware is everything you can touch. Software is every set of instructions that tells hardware what to do.

Think of hardware as a piano and software as sheet music. The piano (hardware) can make sound, but it needs the music (software) to tell it which keys to press and when. Without software, hardware does nothing. Without hardware, software has nothing to run on.

  • CPU — the processor chip inside your computer
  • RAM sticks — the memory modules plugged into the motherboard
  • SSD / HDD — the physical drive that stores your files
  • Keyboard & mouse — physical input devices
  • Monitor — the physical screen
  • Motherboard — the circuit board that connects everything

System software manages the hardware and provides a platform for application software to run on.

  • Operating System (OS) — Windows, macOS, Linux (see next section!)
  • Device drivers — software that lets your OS talk to hardware like printers
  • BIOS / UEFI — the very first software that runs when you press the power button

Application software is built on top of the operating system and is what most people think of when they hear "software."

  • Web browsers — Chrome, Firefox, Safari
  • Office apps — Microsoft Word, Google Docs
  • Media players — Spotify, VLC
  • Games — any game you install and play
  • This website — an app that runs inside your browser

Operating Systems

An operating system (OS) is the master program that manages all the hardware and lets you run other software.

Without an OS your computer would be a blank, unresponsive box. The OS starts up when you press power, shows you the desktop, manages files, and makes sure programs can share the CPU and RAM fairly. The three most common operating systems are below.

Windows

Made by Microsoft. The most widely used OS in the world — you'll find it on the majority of office and home PCs. Great software support and very beginner-friendly.

  • Cost: Paid (often included with new PCs)
  • Market share: ~72% of desktops
  • Best for: Everyday use, gaming, business

macOS

Made by Apple, exclusively for Mac computers. Known for its sleek design, stability, and tight integration with iPhones and iPads. Popular with creative professionals.

  • Cost: Free (comes with Mac hardware)
  • Market share: ~16% of desktops
  • Best for: Design, video editing, Apple users
Free & Open Source

Linux

Built by a global community of volunteers. Completely free and open-source — anyone can view, modify, and share the code. Powers most of the world's servers and the Android OS.

  • Cost: Free
  • Market share: ~4% desktop, ~96% servers
  • Best for: Developers, servers, privacy
